Install as PWA

  1. Chrome/Edge: Click "Install App" button or look for install icon in address bar
  2. Mobile Safari: Tap Share button → "Add to Home Screen"
  3. Android Chrome: Tap menu (⋮) → "Add to Home screen"
  4. Once installed, the app will work offline and can be launched like a native app
